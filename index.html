Evento de Programação de Python

Olá, seja bem-vindo(a) à página do evento de programação de Python, um evento online e gratuito que vai acontecer no dia 10 de outubro de 2023, das 9h às 17h. O objetivo deste evento é promover o aprendizado e o compartilhamento de conhecimentos sobre a linguagem Python, uma das mais populares e versáteis do mundo. O evento é destinado a todos os interessados em programação, desde iniciantes até profissionais.

A programação do evento é a seguinte:

  • 9h - Abertura e boas-vindas
  • 9h30 - Palestra: Introdução ao Python
  • 10h30 - Intervalo
  • 11h - Oficina: Instalação e configuração do ambiente Python
  • 12h - Almoço
  • 13h - Palestra: Estruturas de dados e algoritmos em Python
  • 14h - Oficina: Resolução de problemas com Python
  • 15h - Intervalo
  • 15h30 - Palestra: Aplicações web com Python
  • 16h30 - Oficina: Desenvolvimento de um site simples com Python
  • 17h - Encerramento e agradecimentos
