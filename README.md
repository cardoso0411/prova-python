# Prova Python

Este repositório foi criado como parte de uma prova para a disciplina de **Desenvolvimento Web 1** no primeiro semestre do curso de **Desenvolvimento de Software Multiplataforma (DSM)** da **FATEC Itapira**.

O objetivo da atividade foi aplicar os conhecimentos adquiridos em HTML e CSS, desenvolvendo páginas web para um evento fictício de programação em Python. O projeto inclui páginas de apresentação do evento, formulário de inscrição, informações sobre Python e confirmação de participação.
